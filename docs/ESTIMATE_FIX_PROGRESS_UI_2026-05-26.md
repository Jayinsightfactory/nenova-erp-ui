# 견적서관리 출고확정 진행 표시 UI (2026-05-26)

## 요청

견적서관리에서 일괄 확정을 누르면 실제로 확정이 진행 중인지, 얼마나 남았는지 눈으로 볼 수 있게 로딩 화면을 표시한다.

## 반영 내용

- `pages/estimate.js`
  - `fixProgress` 상태 추가
  - 사전검증 단계에서 현재 검증 중인 세부차수와 진행률 표시
  - 실제 확정 단계에서 현재 확정 중인 세부차수, 완료/남은 개수, 성공/실패 개수 표시
  - 최근 처리 결과 일부를 하단에 표시

## 목적

- `usp_ShipmentFix`, `usp_StockCalculation` 실행 중 화면이 멈춘 것처럼 보이지 않게 함
- deadlock 자동 재시도나 재고계산으로 시간이 걸릴 때 사용자가 현재 진행 상태를 확인할 수 있게 함
- 확정 완료 후 기존 결과 모달은 그대로 유지
