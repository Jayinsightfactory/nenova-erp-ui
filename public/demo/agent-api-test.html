에이전트 API 테스트 콘솔

"모든 것은 API로 돌아간다"를 증명하는 페이지 — 에이전트가 쓰는 것과 동일한 엔드포인트를 사람이 직접 호출해봅니다.

⚠ 이 콘솔은 읽기(GET) 전용입니다. 쓰기 API는 에이전트도 "검증 대기 → 사람 컨펌" 게이트를 거치므로 시연은 분배/주문 화면에서 진행하세요. 호출에는 로그인 세션(JWT)이 필요합니다 — 먼저 /login 후 사용.

1. 엔드포인트 선택 → 호출

에이전트(n8n, Claude, 외부 자동화)는 이 동일한 URL을 토큰 인증으로 호출합니다 — 사람용 화면과 에이전트용 통로가 같습니다.

2. 응답

상태: —응답 시간: —크기: —
아직 호출 전입니다. 위에서 엔드포인트를 선택하고 [호출]을 누르세요.

3. 에이전트 연동 안내 (영업 시연 멘트)
